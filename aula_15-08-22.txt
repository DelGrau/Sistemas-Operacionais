Microsoft
    md <nome> (Abreviação para 'mkdir') - Cria uma pasta
    cd <nome> - Vai para uma pasta
    type <nome> - Lista o que tem dentro do arquivo
    del <nome>
    rmdir <nome>
    cls - Limpar tela
    dir - Lista as coisas dentro da pasta

    Criar um olá mundo
        cd <pasta>
        qbasic olamundo.bas - cria um arquivo em QBASIC, linguagem própria do MS-DOS
        print "Ola Mundo"
        SHIFT+F5
